📦 Парсер и загрузчик данных олимпиады

Этот проект представляет собой Python-скрипт для обработки, анализа и загрузки данных о результатах студентов.  
Он автоматизирует перенос данных из файла (CSV/Excel) в базу данных PostgreSQL и скачивает все прикреплённые файлы по ссылкам из датасета.

## 🚀 Основные возможности

- **Чтение данных** — из `.csv` или `.xlsx` файлов.  
- **Очистка данных** — автоматическое переименование колонок на английский язык, обработка пустых значений (`NULL`, `#N/A`).  
- **Загрузка в БД** — подключение к PostgreSQL, создание таблицы и загрузка очищенных данных.  
- **Скачивание файлов** — поиск всех ссылок в данных и загрузка прикреплённых файлов (изображения, документы) в локальную папку.

## ⚙️ Требования

Перед запуском убедитесь, что у вас установлено:

1. **Python 3.7+**  
2. **PostgreSQL Server** — установленный и запущенный сервер баз данных.  
3. **Библиотеки Python** — все необходимые зависимости перечислены в `requirements.txt`.

## 🛠️ Установка и настройка

### 1. Клонирование или скачивание репозитория

```bash
git clone <URL вашего репозитория>
cd <название папки проекта>
```

> Либо просто скачайте все файлы в одну папку.

### 2. Создание базы данных

Скрипт **не создаёт** саму базу данных, только таблицу в ней.  
Вручную создайте пустую БД в PostgreSQL (через `psql` или `pgAdmin`):

```sql
CREATE DATABASE sirius_db;
```

### 3. Установка зависимостей

В терминале (в папке проекта) выполните:

```bash
pip install -r requirements.txt
```

### 4. Настройка скрипта (важно!)

Откройте `main.py` и отредактируйте блок **НАСТРОЙКИ** в самом верху:

```python
# Подключение к PostgreSQL
DB_USER     = 'postgres'      # Ваше имя пользователя
DB_PASSWORD = 'password'      # ВАШ ПАРОЛЬ
DB_HOST     = 'localhost'     # Хост (обычно localhost)
DB_PORT     = '5432'          # Порт (обычно 5432)
DB_NAME     = 'sirius_db'     # Имя базы данных (создали на шаге 2)
TABLE_NAME  = 'student_tasks' # Имя таблицы

# Настройки файлов
DATASET_PATH = 'data.csv'     # Путь к вашему файлу с данными
```

> **Примечание:** если файл в формате `.xlsx`, измените `DATASET_PATH` на `data.xlsx` и в коде замените `pd.read_csv` на `pd.read_excel`.

### 5. Размещение файла данных

Положите ваш файл с данными (например, `data.csv`) в ту же папку, где находится `main.py`.

## ▶️ Запуск скрипта

```bash
python main.py
```

В консоли появится лог выполнения:  
- чтение файла  
- сохранение в БД  
- процесс скачивания файлов  

По завершении работы рядом со скриптом появится папка `downloads` со всеми скачанными файлами.

## ⚠️ Важные замечания

- **Перезапись таблицы**  
  Текущая конфигурация (`if_exists='replace'`) **полностью удаляет и пересоздаёт** таблицу при каждом запуске.  
  Если нужно добавлять данные, измените значение на `'append'`.

- **Ошибка авторизации (403 Forbidden)**  
  Если при скачивании файлов возникают ошибки с кодом 403 или 401, сервер требует авторизации.  
  Для обхода может потребоваться передать cookies вашей браузерной сессии в `requests.get()`.
